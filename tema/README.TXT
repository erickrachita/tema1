librariile instalate: pandas si matplotlib
modul de instalare: am intrat in command prompt si am folosit comenzile:
pip install pandas
pip install matplotlib

mi-a fost recomandata versiunea 3.9.10 a python-ului, asa ca aceasta este cea folosita in IDLE.
